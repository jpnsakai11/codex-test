# NovaSearch

A lightweight search engine style app inspired by Google's clean search experience.

## Features

- Centered homepage with a search box and two action buttons.
- Relevance scoring over a local index of sample websites.
- Search results list with title, URL, and snippet.
- "I'm Feeling Lucky" behavior that opens the top result.

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173` in your browser.
